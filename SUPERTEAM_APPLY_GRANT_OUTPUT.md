## Step 2: Details

**Project Details**

> Solana agents can prepare transactions faster than users can inspect them, leaving a gap between agent intent, policy compliance, and durable evidence. IntentProof is a local-first control layer for agent-proposed SOL transfers: it checks a recipient allowlist, per-action and daily limits, and a required intent note, then records every allow or block decision in an SHA-256-linked receipt chain. For an allowed action, the user signs from Phantom on Devnet; the app never holds keys. The MVP is already deployed. This grant completes a reproducible Devnet walkthrough and release notes.
